Scripts y Comandos

Scripts y comandos útiles para la gestión de VPS.

Comando de Instalación Remota:

# Copiar script al VPS scp setup-vps.sh root@IP_DEL_VPS:/tmp/ # Conectar y ejecutar ssh root@IP_DEL_VPS chmod +x /tmp/setup-vps.sh /tmp/setup-vps.sh --client "Juan Perez" --domain juanperez.com --type react --email admin@juanperez.com --password secret123

Verificación Post-Instalación:

# Verificar servicios systemctl status nginx systemctl status certbot.timer # Verificar SSL curl -I https://dominio.com # Ver logs tail -f /var/www/dominio.com/logs/access.log

Comandos de Mantenimiento:

# Backup manual /home/cliente/backup.sh # Renovar SSL certbot renew --dry-run # Reiniciar servicios systemctl restart nginx
